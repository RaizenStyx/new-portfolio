# Connor Reed | Full-Stack Developer Portfolio

A professional portfolio website showcasing projects in Shopify Development, React/Next.js Architecture, and Game Development.

Built with **Next.js 16 (App Router)**, **TypeScript**, and **Tailwind CSS**.
